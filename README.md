## Build Setup

``` bash
# install dependencies
$ npm install


#setup html
```
Createa folder dist , in dist folder create html set div id app , and call bundle.js script on footer
```

# serve with hot reload at localhost:8100
$ npm run dev

# build for production and launch server
$ npm run prod
$ npm run start

```
